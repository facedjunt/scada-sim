# SCADA-SIM

**Учебный проект, полученный мною на предприятии**  
REST сервис с имитацией "живых" данных

## О проекте

Разработка, цель которой - создание прототипа SCADA-системы.
Проект имитирует работу промышленных датчиков и отображает их показания в реальном времени.

**Стек технологий:**
- *Бэкенд:* Python (Flask) + PostgreSQL
- *Фронтенд:* JavaScript (Vue) + CSS
- *API:* REST (взаимодействие между клиентом и сервером)

## Планируемый функционал

- Имитация "живых" данных (температура, давление, частота)
- REST API для передачи данных
- Веб-интерфейс с индикаторами
